# NeuraReport Feature Roadmap

Selected features for the document-centric workspace vision.

---

## Document Ingestion & Import

- Drag-drop any file type with auto-detection and preview
- Email-to-document: forward emails to a unique inbox address → becomes a doc
- Web clipper browser extension (save pages as editable docs)
- Screenshot-to-document (OCR + structure detection)
- Voice memo transcription import
- Scan physical documents via phone camera companion app
- Import from Google Docs, Notion, Confluence, Dropbox Paper via OAuth
- Watch folders: auto-import anything dropped in a local folder
- Import Figma/Sketch designs as embedded objects
- Audio/video file import with transcript generation
- Import structured data (JSON, XML, YAML) as editable tables
- Bulk import from ZIP archives with folder structure preservation

---

## AI-Powered Reading & Comprehension

- One-click document summarization (multiple lengths)
- "Explain like I'm 5" mode for complex documents
- Auto-generate executive summary for any document
- Key points extraction with confidence scores
- Summarize changes between document versions in plain English

---

## AI Writing & Editing

- Continue writing from cursor position
- Expand bullet points into full paragraphs
- Compress paragraphs into bullet points
- Auto-fix grammar and spelling with explanations
- Write section from outline

---

## Smart Templates & Document Generation

- Template library with 1000+ pre-built templates
- AI-generated templates from description ("I need a consulting proposal")
- Template variables with conditional logic
- Template inheritance (child templates override parent sections)
- Dynamic sections that appear/hide based on data
- Mail merge from any data source
- Generate personalized document batches
- Template versioning with diff view
- Template analytics: which templates are used most
- Template recommendations based on document type
- Clone any document as a template
- Template fragments: reusable blocks across templates
- Generate templates from example documents
- Industry-specific template packs

## Spreadsheet & Data Features

- Natural language queries on spreadsheet data
- AI formula assistant: describe what you want, get formula
- Spreadsheet-to-chart with one click
- Generate synthetic test data
- Import/export Excel, CSV, Google Sheets
- Undo/redo with branching
- Spreadsheet templates for budgets, invoices, trackers
- Form-to-spreadsheet: generate input forms from sheet schema
- Spreadsheet validation against schema
